# SauceDemo Automation Testing

Automation Testing menggunakan:

- Selenium WebDriver
- Mocha
- Mochawesome
- Page Object Model (POM)
- Visual Regression

## Test Case

### Positive

- Login Success
- Add To Cart
- Checkout

### Negative

- Invalid Username
- Wrong Password
- Locked Out User

## Menjalankan Test

```bash
npm test
```
